# UI 개발용 임시 폴더

이 폴더는 UI 작업을 위한 임시 개발 환경입니다.

## 특징

- **Mock 데이터 사용**: 서버 없이도 UI를 테스트할 수 있습니다
- **임의 드론 데이터 포함**: 3개의 샘플 드론이 미리 등록되어 있습니다
- **독립 실행**: 원본 프로젝트와 분리되어 있어 안전하게 UI를 수정할 수 있습니다

## 포함된 Mock 데이터

### 드론 목록
- `drone_alpha` - 활성 상태, 화재 감지 로그 3개
- `drone_beta` - 활성 상태, 화재 감지 로그 1개  
- `drone_gamma` - 오프라인 상태, 로그 없음

### 화재 감지 로그
- 각 드론별로 다양한 확률의 감지 로그 포함
- 이미지 URL은 placeholder 사용

## 실행 방법

```bash
cd frontend_ui_dev
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## Mock 데이터 수정

`mockData.js` 파일을 수정하여 원하는 드론 데이터를 추가/수정할 수 있습니다.

## 주의사항

- 이 폴더의 변경사항은 원본 프로젝트에 영향을 주지 않습니다
- UI 작업이 완료되면 변경사항을 원본 프로젝트에 반영해야 합니다
